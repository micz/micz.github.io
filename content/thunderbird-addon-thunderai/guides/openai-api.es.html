La integración con la OpenAI API conecta ThunderAI directamente con los servidores de OpenAI mediante una clave de API.
En comparación con la interfaz web, es más rápida, más fiable y te da control explícito sobre qué modelo usar.

Ideal para Acceso estable y rápido a los modelos de ChatGPT

Requisitos previos

  • Una cuenta de OpenAI con acceso a la API habilitado
  • Una clave de API válida
  • Thunderbird con ThunderAI instalado

Nota: El uso de la API de OpenAI se factura por separado de una suscripción a ChatGPT. Aunque tengas ChatGPT Plus, igualmente necesitas añadir créditos a tu cuenta de API.

Cómo obtener una clave de API

  1. Ve a platform.openai.com/api-keys
  2. Haz clic en Create new secret key
  3. Dale un nombre (p. ej. "ThunderAI") y haz clic en Create
  4. Copia la clave inmediatamente; no se mostrará de nuevo

Configuración

1. Abre las Opciones de ThunderAI

En Thunderbird, haz clic en el menú de ThunderAI y selecciona Options, o ve a
Herramientas → Complementos y temas → ThunderAI → Preferencias.

2. Selecciona la integración OpenAI API

En la sección Connection, elige OpenAI API del menú desplegable de integración.

3. Introduce tu clave de API

Pega tu clave de API en el campo API Key. La clave se almacena localmente en Thunderbird y nunca se envía a ningún lugar salvo a los servidores de OpenAI.

4. Elige un modelo

Selecciona el modelo que deseas usar del menú desplegable Model. Haz clic en el botón para actualizar la lista.

5. Guarda y prueba

Haz clic en Save. Abre cualquier correo electrónico, usa el menú de ThunderAI y ejecuta un prompt para verificar que todo funciona.

Consejos

  • El uso de la API se mide en tokens. Los correos suelen ser cortos, por lo que los costes son generalmente muy bajos. Consulta tu uso en platform.openai.com/usage.
  • OpenAI añade nuevos modelos regularmente. Actualiza la lista de modelos en las opciones de ThunderAI para ver los últimos disponibles.

Solución de problemas

Error "Invalid API key"

Verifica que copiaste la clave correctamente — no debería haber espacios adicionales. Si la clave sigue fallando, genera una nueva desde la plataforma de OpenAI.

Error "You exceeded your current quota"

Tu cuenta de la API de OpenAI se ha quedado sin créditos. Añade más en platform.openai.com/settings/organization/billing.

No aparecen modelos en el menú desplegable

Asegúrate de que tu clave de API está guardada y es válida. Si el problema persiste, comprueba tu conexión a internet.
