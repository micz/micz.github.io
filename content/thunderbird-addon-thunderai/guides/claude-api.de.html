Die Integration mit der Claude API verbindet ThunderAI mit den Claude-KI-Modellen von Anthropic.
Claude ist bekannt für sorgfältig strukturierte Antworten und die effiziente Verarbeitung langer Dokumente — ideal zum Lesen und Verfassen detaillierter E-Mails.

API-Schlüssel erforderlich Ja
Kostenlose Option Nein [Nutzungsbasierte Abrechnung]
Geeignet für Detaillierte Analysen, lange E-Mails, strukturiertes Schreiben

Voraussetzungen

  • Ein Anthropic-Konto mit API-Zugang
  • Ein gültiger Claude API-Schlüssel
  • Thunderbird mit installiertem ThunderAI

API-Schlüssel erhalten

  1. Gehen Sie zu console.anthropic.com/settings/keys
  2. Klicken Sie auf Create Key
  3. Geben Sie ihm einen Namen und kopieren Sie den Schlüssel sofort — er wird nicht noch einmal angezeigt

Einrichtung

1. ThunderAI-Optionen öffnen

Klicken Sie in Thunderbird auf das ThunderAI-Menü und wählen Sie Options, oder gehen Sie zu
Extras → Add-ons und Themes → ThunderAI → Einstellungen.

2. Claude API-Integration auswählen

Wählen Sie im Bereich Connection die Option Claude API aus dem Integrations-Dropdown.

3. API-Schlüssel eingeben

Fügen Sie Ihren Claude API-Schlüssel in das Feld API Key ein.

4. Erforderliche Berechtigung erteilen

Die Claude API erfordert, dass ThunderAI auf die Domain anthropic.com zugreift.
Auf der Optionsseite erscheint eine Schaltfläche, um diese Berechtigung zu erteilen — klicken Sie darauf und bestätigen Sie im Thunderbird-Dialog.

Dies ist eine optionale Berechtigung, die nur angefordert wird, wenn Sie die Claude-Integration auswählen. Sie ermöglicht ThunderAI die Kommunikation mit den API-Servern von Anthropic. ThunderAI erfasst keine Daten.

5. Modell auswählen

Wählen Sie ein Claude-Modell aus dem Dropdown Model. Klicken Sie auf die Schaltfläche, um die Liste zu aktualisieren.

6. Speichern und testen

Klicken Sie auf Save. Öffnen Sie eine beliebige E-Mail, verwenden Sie das ThunderAI-Menü und führen Sie einen Prompt aus, um zu bestätigen, dass die Integration funktioniert.

Tipps

  • Claude-Modelle verarbeiten dank ihres erweiterten Kontextfensters große Textmengen gut — nützlich bei langen E-Mail-Threads.
  • Claude neigt dazu, ausführliche Antworten zu geben. Wenn Sie kürzere Ausgaben bevorzugen, fügen Sie Anweisungen wie „Sei prägnant" direkt in Ihre Prompts oder in eine benutzerdefinierte Prompt-Vorlage ein.
  • Überwachen Sie Ihre API-Nutzung und Kosten auf console.anthropic.com.

Fehlerbehebung

„Authentication error" oder „Invalid API key"

Prüfen Sie, ob der Schlüssel korrekt kopiert wurde. Generieren Sie bei Bedarf einen neuen Schlüssel über die Anthropic-Konsole.

CORS-Fehler oder fehlschlagende Anfragen

Stellen Sie sicher, dass Sie die Berechtigung für die Domain anthropic.com wie in Schritt 4 beschrieben erteilt haben. Ohne diese Berechtigung kann ThunderAI die Claude API nicht erreichen.

Fehler „Credit balance too low"

Fügen Sie Ihrem Anthropic-Konto Guthaben unter console.anthropic.com/settings/billing hinzu.

Keine Modelle im Dropdown sichtbar

Prüfen Sie, ob Ihr API-Schlüssel gespeichert und die Domain-Berechtigung erteilt wurde. Versuchen Sie, die Optionsseite zu speichern und erneut zu öffnen.
