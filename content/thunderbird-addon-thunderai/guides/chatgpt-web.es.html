La integración con la ChatGPT Web Interface te permite usar ThunderAI sin una clave de API.
ThunderAI interactúa directamente con el sitio web de ChatGPT en tu nombre, por lo que solo necesitas una cuenta gratuita de ChatGPT.

Clave de API requerida No
Opción gratuita Sí [También con una cuenta gratuita de ChatGPT]
Ideal para Empezar rápidamente, sin configuración de facturación

Requisitos previos

  • Una cuenta gratuita (o de pago) en chatgpt.com
  • Thunderbird con ThunderAI instalado

Configuración

1. Abre las Opciones de ThunderAI

En Thunderbird, haz clic en el menú de ThunderAI y selecciona Options, o ve a
Herramientas → Complementos y temas → ThunderAI → Preferencias.

2. Selecciona la ChatGPT Web Interface

En la sección Connection, elige ChatGPT Web Interface del menú desplegable de integración.

3. Concede el permiso requerido

ThunderAI necesita permiso para acceder a chatgpt.com.

Este permiso es opcional y solo se usa para interactuar con el sitio web de ChatGPT en tu nombre. ThunderAI no recopila ni almacena ninguno de tus datos.

4. Inicia sesión en ChatGPT desde Thunderbird

Solo necesitas hacerlo una vez; Thunderbird mantendrá la sesión activa.
Usa el botón en la página de preferencias de ThunderAI para abrir una nueva pestaña de Thunderbird y visitar chatgpt.com. Inicia sesión con tu cuenta.

5. Prueba la integración

Abre cualquier correo electrónico, haz clic en el botón de la barra de ThunderAI y selecciona un prompt (p. ej. Summarize). ThunderAI abrirá un panel e interactuará con ChatGPT.

Consejos y limitaciones conocidas

  • La velocidad de respuesta depende del sitio web de ChatGPT y puede variar. La integración mediante API suele ser más rápida y fiable.
  • Las cuentas gratuitas están sujetas a los límites de uso propios de ChatGPT (límites de frecuencia, disponibilidad de modelos).
  • Si ChatGPT cambia el diseño de su sitio web, la integración puede dejar de funcionar temporalmente hasta que ThunderAI sea actualizado. Consulta la página de estado para conocer los problemas conocidos.
  • Puedes definir un Custom GPT o Project de forma global en la ventana de preferencias o en la configuración de cada prompt individual.

Solución de problemas

ThunderAI muestra un error o no hay respuesta

Asegúrate de haber iniciado sesión en ChatGPT en una pestaña de Thunderbird. Intenta visitar chatgpt.com directamente desde Thunderbird y comprueba que la sesión está activa. Usa el botón en la página de preferencias de ThunderAI para hacerlo.

Las respuestas son lentas o agotan el tiempo

Suele ser un problema temporal del sitio web de ChatGPT. Espera un momento e inténtalo de nuevo, o considera cambiar a la integración OpenAI API para un rendimiento más fiable.
