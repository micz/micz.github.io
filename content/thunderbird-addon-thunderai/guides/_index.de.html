KI-Integrationen

Verbinden Sie ThunderAI mit dem KI-Backend Ihrer Wahl. Sie wissen nicht, welches Sie wählen sollen? Sehen Sie sich die Vergleichstabelle unten an.

  • ChatGPT Web Interface — Kein API-Schlüssel erforderlich, funktioniert auch mit einem kostenlosen ChatGPT-Konto.
  • OpenAI API — Direkte Verbindung zu ChatGPT-Modellen über einen API-Schlüssel.
  • Google Gemini — Verbinden Sie sich mit Googles Gemini-Modellen, kostenloses Kontingent verfügbar.
  • Claude API — Anthropics Claude-Modelle, ideal für die detaillierte E-Mail-Analyse.
  • Ollama (Lokale KI) — Führen Sie KI vollständig auf Ihrem Computer aus. Privat, kostenlos, kein Internet erforderlich.
  • OpenAI Compatible API — LM Studio, DeepSeek, Mistral, OpenRouter und mehr.
