Intégrations AI

Connectez ThunderAI au backend AI de votre choix. Vous ne savez pas lequel choisir ? Consultez le tableau comparatif ci-dessous.

  • ChatGPT Web Interface — Aucune clé API requise, fonctionne aussi avec un compte ChatGPT gratuit.
  • OpenAI API — Connexion directe aux modèles ChatGPT via clé API.
  • Google Gemini — Connectez-vous aux modèles Gemini de Google, niveau gratuit disponible.
  • Claude API — Les modèles Claude d'Anthropic, idéaux pour l'analyse détaillée des e-mails.
  • Ollama (IA locale) — Exécutez l'IA entièrement sur votre machine. Privé, gratuit, sans internet.
  • OpenAI Compatible API — LM Studio, DeepSeek, Mistral, OpenRouter et bien d'autres.

Vous voulez simplement essayer ThunderAI gratuitement ? Commencez avec la ChatGPT Web Interface, aucune configuration, aucune facturation.
Vous tenez à votre vie privée ? Utilisez Ollama pour garder tout sur votre machine.
