The ChatGPT Web Interface integration lets you use ThunderAI without an API key.
ThunderAI interacts directly with the ChatGPT website on your behalf, so a free ChatGPT account is all you need.

API key required No
Free option Yes — a free ChatGPT account is enough
Best for Getting started quickly, no billing setup

Prerequisites

  • A free (or paid) account on chatgpt.com
  • Thunderbird with ThunderAI installed

Setup

1. Open ThunderAI Options

In Thunderbird, click the ThunderAI menu and select Options, or go to
Tools → Add-ons and Themes → ThunderAI → Preferences.

2. Select the ChatGPT Web Interface

In the Connection section, choose ChatGPT Web Interface from the integration dropdown.

3. Grant the required permission

ThunderAI needs permission to access chatgpt.com.
A button will appear on the options page, click it and confirm in the Thunderbird permission dialog.

This permission is optional and only used to interact with the ChatGPT website on your behalf. ThunderAI does not collect or store any of your data.

4. Log in to ChatGPT in Thunderbird

Open a new Thunderbird tab and visit chatgpt.com. Log in with your account.
You only need to do this once, Thunderbird will keep the session active. Use the button in the ThunderAI preferences page to do this.

5. Test the integration

Open any email, right-click or use the ThunderAI toolbar button, and select a prompt (e.g. Summarize). ThunderAI will open a panel and interact with ChatGPT in the background.

Tips and known limitations

  • Response speed depends on the ChatGPT website and may vary. The API integration is generally faster and more reliable.
  • Free accounts are subject to ChatGPT's own usage limits (rate limits, model availability).
  • If ChatGPT changes its website layout, the integration may temporarily stop working until ThunderAI is updated. Check the status page for any known issues.
  • If you have a ChatGPT Plus subscription, the web interface will automatically use GPT-4 class models.

Troubleshooting

ThunderAI shows an error or no response

Make sure you are logged in to ChatGPT in a Thunderbird tab. Try visiting chatgpt.com directly from Thunderbird and check that the session is active. Use the button in the ThunderAI preferences page to do this.

The permission button does not appear

Try closing and reopening the ThunderAI options page.

Responses are slow or time out

This is usually a temporary issue with the ChatGPT website. Wait a moment and try again, or consider switching to the OpenAI API integration for more reliable performance.
