L'intégration OpenAI API connecte ThunderAI directement aux serveurs d'OpenAI via une clé API.
Par rapport à l'interface web, elle est plus rapide, plus fiable et vous donne un contrôle explicite sur le modèle utilisé.

Clé API requise Oui
Option gratuite Non [Facturation à l'usage]
Idéal pour Accès stable et rapide aux modèles ChatGPT

Prérequis

  • Un compte OpenAI avec accès à l'API activé
  • Une clé API valide
  • Thunderbird avec ThunderAI installé

Note : l'utilisation de l'API OpenAI est facturée séparément d'un abonnement ChatGPT. Même si vous avez ChatGPT Plus, vous devez quand même ajouter des crédits à votre compte API.

Obtenir une clé API

  1. Accédez à platform.openai.com/api-keys
  2. Cliquez sur Create new secret key
  3. Donnez-lui un nom (ex. : « ThunderAI ») et cliquez sur Create
  4. Copiez la clé immédiatement, elle ne sera plus affichée

Configuration

1. Ouvrir les options de ThunderAI

Dans Thunderbird, cliquez sur le menu ThunderAI et sélectionnez Options, ou accédez à
Outils → Modules complémentaires et thèmes → ThunderAI → Préférences.

2. Sélectionner l'intégration OpenAI API

Dans la section Connection, choisissez OpenAI API dans le menu déroulant des intégrations.

3. Saisir votre clé API

Collez votre clé API dans le champ API Key. La clé est stockée localement dans Thunderbird et n'est jamais envoyée ailleurs qu'aux serveurs d'OpenAI.

4. Choisir un modèle

Sélectionnez le modèle que vous souhaitez utiliser dans le menu déroulant Model. Cliquez sur le bouton pour actualiser la liste.

5. Enregistrer et tester

Cliquez sur Save. Ouvrez ensuite n'importe quel e-mail, utilisez le menu ThunderAI et exécutez un prompt pour vérifier que tout fonctionne.

Conseils

  • L'utilisation de l'API est mesurée en tokens. Les e-mails sont généralement courts, donc les coûts sont habituellement très faibles. Vérifiez votre utilisation sur platform.openai.com/usage.
  • OpenAI ajoute régulièrement de nouveaux modèles. Actualisez la liste des modèles dans les options ThunderAI pour voir les derniers disponibles.

Dépannage

Erreur « Invalid API key »

Vérifiez que vous avez bien copié la clé — il ne doit pas y avoir d'espaces supplémentaires. Si la clé échoue encore, générez-en une nouvelle depuis la plateforme OpenAI.

Erreur « You exceeded your current quota »

Votre compte API OpenAI n'a plus de crédits. Ajoutez-en sur platform.openai.com/settings/organization/billing.

Aucun modèle n'apparaît dans le menu déroulant

Assurez-vous que votre clé API est enregistrée et valide. Si le problème persiste, vérifiez votre connexion internet.
