La integración con Google Gemini conecta ThunderAI con los modelos de IA Gemini de Google a través de la API de Gemini.
Incluye soporte para opciones avanzadas como System Instructions y thinkingBudget, útiles para usuarios avanzados que quieren mayor control sobre el comportamiento del modelo.

Clave de API requerida Sí
Opción gratuita Sí [Nivel gratuito generoso disponible]
Ideal para Usuarios del ecosistema de Google o que buscan una opción de API gratuita

Requisitos previos

  • Una cuenta de Google
  • Una clave de API de Gemini (nivel gratuito disponible)
  • Thunderbird con ThunderAI instalado

Cómo obtener una clave de API

  1. Ve a aistudio.google.com/apikey
  2. Haz clic en Create API key
  3. Selecciona un proyecto de Google Cloud existente o crea uno nuevo
  4. Copia la clave de API generada

La API de Gemini tiene un nivel gratuito con límites de frecuencia generosos, adecuado para uso personal con correos electrónicos. Hay niveles de pago disponibles si necesitas mayor rendimiento.

Configuración

1. Abre las Opciones de ThunderAI

En Thunderbird, haz clic en el menú de ThunderAI y selecciona Options, o ve a
Herramientas → Complementos y temas → ThunderAI → Preferencias.

2. Selecciona la integración Google Gemini

En la sección Connection, elige Google Gemini del menú desplegable de integración.

3. Introduce tu clave de API

Pega tu clave de API en el campo API Key.

4. Elige un modelo

Selecciona el modelo Gemini del menú desplegable Model. Haz clic en el botón para actualizar la lista.

5. (Opcional) Configura las instrucciones del sistema

Las instrucciones del sistema te permiten establecer un contexto persistente para todos tus prompts. Por ejemplo, indicarle al modelo que siempre responda formalmente, en un idioma específico o que asuma un determinado rol.
Introduce tus instrucciones en el campo System Instructions de la página de opciones.

6. (Opcional) Configura thinkingBudget

La opción thinkingBudget controla cuánto razonamiento interno realiza el modelo antes de responder. Un valor más alto puede mejorar la calidad en tareas complejas a costa de velocidad y cuota. Déjalo en el valor predeterminado a menos que tengas necesidades específicas.

7. Guarda y prueba

Haz clic en Save. Abre cualquier correo electrónico, usa el menú de ThunderAI y ejecuta un prompt para verificar la conexión.

Consejos

  • El nivel gratuito se renueva mensualmente y es generalmente suficiente para uso personal.
  • Las instrucciones del sistema son especialmente útiles si necesitas especificar reglas generales para tu asistente de correo electrónico.

Solución de problemas

Error "API key not valid"

Verifica que la clave fue copiada correctamente. Si es necesario, genera una nueva desde AI Studio.

"Model not found" o lista de modelos vacía

Asegúrate de que tu clave de API tiene acceso a los modelos Gemini. Algunas claves creadas en proyectos de Google Cloud con restricciones pueden tener acceso limitado a los modelos.
