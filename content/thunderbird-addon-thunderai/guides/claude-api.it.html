L'integrazione con la Claude API connette ThunderAI ai modelli AI Claude di Anthropic.
Claude è noto per produrre risposte accurate e ben strutturate, gestendo ottimamente i documenti lunghi — ideale per leggere e comporre email dettagliate.

Chiave API richiesta Sì
Opzione gratuita No [Pagamento a consumo]
Ideale per Analisi dettagliate, email lunghe, scrittura strutturata

Prerequisiti

  • Un account Anthropic con accesso alle API
  • Una chiave Claude API valida
  • Thunderbird con ThunderAI installato

Ottenere una chiave API

  1. Vai su console.anthropic.com/settings/keys
  2. Clicca su Create Key
  3. Assegnagli un nome e copia subito la chiave: non verrà mostrata di nuovo

Configurazione

1. Apri le opzioni di ThunderAI

In Thunderbird, clicca sul menu ThunderAI e seleziona Options, oppure vai su
Strumenti → Componenti aggiuntivi e temi → ThunderAI → Preferenze.

2. Seleziona l'integrazione Claude API

Nella sezione Connection, scegli Claude API dal menu a tendina delle integrazioni.

3. Inserisci la tua chiave API

Incolla la tua chiave Claude API nel campo API Key.

4. Concedi il permesso richiesto

La Claude API richiede che ThunderAI acceda al dominio anthropic.com.
Nella pagina delle opzioni comparirà un pulsante per concedere questo permesso — clicca su di esso e conferma nella finestra di dialogo di Thunderbird.

Si tratta di un permesso opzionale richiesto solo quando selezioni l'integrazione Claude. Consente a ThunderAI di comunicare con i server API di Anthropic. ThunderAI non raccoglie alcun dato.

5. Scegli un modello

Seleziona un modello Claude dal menu a tendina Model. Clicca sul pulsante per aggiornare l'elenco.

6. Salva e testa

Clicca su Save. Apri una qualsiasi email, usa il menu ThunderAI ed esegui un prompt per confermare che l'integrazione funzioni.

Suggerimenti

  • I modelli Claude gestiscono grandi quantità di testo grazie alla loro ampia finestra di contesto, utile quando si elaborano lunghe conversazioni email.
  • Claude tende a essere dettagliato nelle risposte. Se preferisci output più brevi, aggiungi istruzioni come "Sii conciso" direttamente nei tuoi prompt o in un template di prompt personalizzato.
  • Monitora il tuo utilizzo e i costi su console.anthropic.com.

Risoluzione dei problemi

"Authentication error" o "Invalid API key"

Verifica che la chiave sia stata copiata correttamente. Se necessario, rigenera una nuova chiave dalla console Anthropic.

Errore CORS o richieste che falliscono

Assicurati di aver concesso il permesso per il dominio anthropic.com come descritto al passaggio 4. Senza di esso, ThunderAI non può raggiungere la Claude API.

Errore "Credit balance too low"

Aggiungi crediti al tuo account Anthropic su console.anthropic.com/settings/billing.

Nessun modello nel menu a tendina

Verifica che la chiave API sia salvata e che il permesso per il dominio sia stato concesso. Prova a salvare la pagina delle opzioni e riaprirla.
