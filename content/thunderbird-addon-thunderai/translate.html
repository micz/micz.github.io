If you want to help translate {{< param addon_name >}} into a new language or improve an existing translation, you're in the right place!
Your contributions are invaluable in making {{< param addon_name >}} accessible to a wider audience.

On this page, you'll find links for each language, as well as a way to add a new one.

The tool "Hosted Weblate" is easy to use:

  • Click on the language you want to translate or improve to open a new page.
  • On the page you'll find all the strings to be translated.
  • Just translate the strings, press the "Save" buttuon and it's done!


Choose the link for the language you want to translate or improve, or start a new one:
The language status represents the percentage of translated strings in the latest stable release.
