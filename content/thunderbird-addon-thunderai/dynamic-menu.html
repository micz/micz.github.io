You can open the menu clicking with the mouse on the ThunderAI button or using the CRTL+ALT+A keyboard shortcut.

Then, you can interact with the dynamic menu in many ways.
  1. Click with the mouse on a prompt name.
  2. Press a numbered key.
  3. Start writing one or more letter to filter the prompt name.
By default, after selecting a prompt, you need to press Enter to run it.
You can disable this behavior in the options to run the prompt immediately.
