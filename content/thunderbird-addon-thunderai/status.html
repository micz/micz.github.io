Check this page for any known issues in interacting with the ChatGPT Web Interface.


2024/09/23
The ChatGPT web interface is using a new feature that is not compatible with Firefox 115, which causes the ChatGPT Web Interface to stop working in Thunderbird 115.
See #139.

Version 2.1.5 fixes the issue.


Remember that starting from version 2.0.2, to use a response in a reply, the user must select the text.
A single click on the text should suffice, but it is also possible to manually select the desired text.
See #103.
