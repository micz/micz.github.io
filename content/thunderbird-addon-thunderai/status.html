2026/04/21
The ChatGPT web interface has changed and consistently returns an error when attempting to use ThunderAI.
This appears to be an issue with how Thunderbird 140.* loads webpages, and there is no current workaround.
The solution is to update to the monthly version 149.* or wait for the next ESR release scheduled for the beginning of July 2026.
See #776 for more information.
