Privacy

If you're using the ChatGPT web interface or the OpenAI integrations, this add-on connects with OpenAI's API or ChatGPT web interface, sending the prompt on user action, but it does not collect any data itself, other than settings stored locally.
The text is sent to OpenAI's server only on user action and managed by their data retention policy.

  • OpenAI's privacy policy can be found at https://openai.com/policies/privacy-policy
  • OpenAI's API data usage policy is here https://openai.com/policies/api-data-usage-policies


If you're using the Google Gemini integration, this add-on connects with Google Gemini's API, sending the prompt on user action, but it does not collect any data itself, other than settings stored locally.
Google Geminy privacy information can be found at https://safety.google/gemini/


If you're using the Ollama API integration, this add-on connects with the local Ollama instance you configured, sending the prompt on user action, and it does not collect any data itself, other than settings stored locally.


If you're using the generic OpenAI Compatible API integration, this add-on connects with the instance you configured, sending the prompt on user action, and it does not collect any data itself, other than settings stored locally.
The privacy policy depends on the service you're connecting to.


Permissions

These are the permissions used by ThunderAI in Thunderbird.
Remember that ThunderAI operates solely based on user interaction. Aside from automatic tagging and spam detection (if enabled), no actions are performed without direct user input within the add-on menu.

  • Access your data for sites in the chatgpt.com domain: Required to access the ChatGPT website using the web interface. [From version 2.0.5]
  • See your mail accounts, their identities and their folders: Necessary to choose the correct mail identity when replying to an email.
  • Read and modify your email messages as you compose and send them: Required to create a new email message or substitute the text the user has highlighted.
  • Download files and read and modify the browser's download history: Needed for downloading the custom prompts export.
  • Read and modify your email messages as they are displayed to you: Required to create a new email message or substitute the text the user has highlighted.
  • Read your email messages: Required to create a new email message or substitute the text the user has highlighted.
  • Access browser tabs: Required to create a new email message or substitute the text the user has highlighted.
  • [Optional] Create, modify and delete message tags: Required to create new tags when using the "Add tags" option. [From version 3.0.0]
  • [Optional] List message tags: Required to check if the tags received from the API are new or they have to be created, when using the "Add tags" option. [From version 3.0.0]
  • [Optional] Change properties and tags of your email messages: Required to assign tags to an email message when using the "Add tags" option. [From version 3.0.0]
  • [Optional] Move, copy, or delete your email messages: Required to move spam messages to the spam folder, if the automatic spam filter is enabled. [From version 3.2.0]

Removed permissions
  • [Removed in version 2.0.5] Access your data for all websites: Required to access the ChatGPT website using the web interface. This permission was added at the first stages of development and never fixed.
  • [Removed in version 2.1.0] Display notifications to you: The add-on shows a notification if you're trying to use the ChatGPT API integration, but the API key or model are invalid. In such cases, falls back to the web interface, showing a notification with the error. In version 2.1.0 the fallback to the ChatGPT web interface was removed.
