Privacy

This add-on interfaces with OpenAI's API and ChatGPT web interface but collects no data itself other than settings stored locally.
The text is sent to OpenAI's server only on user action and managed by their data retention policy.

  • OpenAI's privacy policy can be found at https://openai.com/policies/privacy-policy
  • OpenAI's API data usage policy is here https://openai.com/policies/api-data-usage-policies


Permissions

These are the Thunderbird permissions used by ThunderAI.
Remember that ThunderAI works only on user interaction, nothing is done without a direct user action on the add-on menu.

  • Access your data for all websites: Needed to access the chatgpt site using the web interface.
  • See your mail accounts, their identities and their folders: Needed to choose the right mail identity when replying to an email.
  • Read and modify your email messages as you compose and send them: Needed to create the mail message or sustitute the text the user has highlighted.
