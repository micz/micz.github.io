+++ Title = "Thunderbird Addon: MS 365 Link Opener" +++ {{< page_title >}}

With this addon you can open a shared link of a SharePoint or OneDrive file directly with the right app, instead of the browser.


How does it work?
After installing, when you open a mail with a link to a shared SharePoint or OneDrive file, in the top right corner of the mail message you'll find a button to open the file directly in the Word, Excel or PowerPoint desktop app.
