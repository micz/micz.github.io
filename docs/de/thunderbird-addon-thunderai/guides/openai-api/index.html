"ThunderAI" Einrichtungsanleitung - OpenAI API

Die Integration mit der OpenAI API verbindet ThunderAI direkt über einen API-Schlüssel mit den OpenAI-Servern.
Im Vergleich zur Web-Oberfläche ist sie schneller, zuverlässiger und gibt Ihnen die explizite Kontrolle darüber, welches Modell Sie verwenden.

API-Schlüssel erforderlichJa
Kostenlose OptionNein [Nutzungsbasierte Abrechnung]
Geeignet fürStabiler, schneller Zugriff auf ChatGPT-Modelle

Voraussetzungen

  • Ein OpenAI-Konto mit aktiviertem API-Zugang
  • Ein gültiger API-Schlüssel
  • Thunderbird mit installiertem ThunderAI

Hinweis: Die Nutzung der OpenAI API wird separat von einem ChatGPT-Abonnement abgerechnet. Auch wenn Sie ChatGPT Plus haben, müssen Sie Ihrem API-Konto dennoch Guthaben hinzufügen.

API-Schlüssel erhalten

  1. Gehen Sie zu platform.openai.com/api-keys
  2. Klicken Sie auf Create new secret key
  3. Geben Sie ihm einen Namen (z. B. „ThunderAI") und klicken Sie auf Create
  4. Kopieren Sie den Schlüssel sofort — er wird nicht noch einmal angezeigt

Einrichtung

1. ThunderAI-Optionen öffnen

Klicken Sie in Thunderbird auf das ThunderAI-Menü und wählen Sie Options, oder gehen Sie zu
Extras → Add-ons und Themes → ThunderAI → Einstellungen.

2. OpenAI API-Integration auswählen

Wählen Sie im Bereich Connection die Option OpenAI API aus dem Integrations-Dropdown.

3. API-Schlüssel eingeben

Fügen Sie Ihren API-Schlüssel in das Feld API Key ein. Der Schlüssel wird lokal in Thunderbird gespeichert und niemals an andere Server als die von OpenAI gesendet.

4. Modell auswählen

Wählen Sie das gewünschte Modell aus dem Dropdown Model. Klicken Sie auf die Schaltfläche, um die Liste zu aktualisieren.

5. Speichern und testen

Klicken Sie auf Save. Öffnen Sie dann eine beliebige E-Mail, verwenden Sie das ThunderAI-Menü und führen Sie einen Prompt aus, um zu überprüfen, dass alles funktioniert.

Tipps

  • Die API-Nutzung wird in Tokens gemessen. E-Mails sind in der Regel kurz, daher sind die Kosten meist sehr gering. Überprüfen Sie Ihre Nutzung unter platform.openai.com/usage.
  • OpenAI fügt regelmäßig neue Modelle hinzu. Aktualisieren Sie die Modellliste in den ThunderAI-Optionen, um die neuesten verfügbaren zu sehen.

Fehlerbehebung

Fehler „Invalid API key"

Prüfen Sie, ob der Schlüssel korrekt kopiert wurde — es sollten keine zusätzlichen Leerzeichen vorhanden sein. Falls der Schlüssel weiterhin fehlschlägt, generieren Sie einen neuen über die OpenAI-Plattform.

Fehler „You exceeded your current quota"

Ihr OpenAI API-Konto hat kein Guthaben mehr. Fügen Sie weiteres Guthaben unter platform.openai.com/settings/organization/billing hinzu.

Keine Modelle im Dropdown sichtbar

Stellen Sie sicher, dass Ihr API-Schlüssel gespeichert und gültig ist. Wenn das Problem weiterhin besteht, prüfen Sie Ihre Internetverbindung.
