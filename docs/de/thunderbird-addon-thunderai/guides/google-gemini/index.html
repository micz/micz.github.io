"ThunderAI" Einrichtungsanleitung - Google Gemini

Die Integration mit Google Gemini verbindet ThunderAI über die Gemini API mit Googles Gemini-KI-Modellen.
Sie unterstützt erweiterte Optionen wie System Instructions und thinkingBudget, die für Nutzer nützlich sind, die das Verhalten des Modells genauer steuern möchten.

API-Schlüssel erforderlichJa
Kostenlose OptionJa [Großzügiges kostenloses Kontingent verfügbar]
Geeignet fürNutzer des Google-Ökosystems oder wer eine kostenlose API-Option sucht

Voraussetzungen

  • Ein Google-Konto
  • Ein Gemini API-Schlüssel (kostenloses Kontingent verfügbar)
  • Thunderbird mit installiertem ThunderAI

API-Schlüssel erhalten

  1. Gehen Sie zu aistudio.google.com/apikey
  2. Klicken Sie auf Create API key
  3. Wählen Sie ein bestehendes Google Cloud-Projekt aus oder erstellen Sie ein neues
  4. Kopieren Sie den generierten API-Schlüssel

Die Gemini API bietet ein kostenloses Kontingent mit großzügigen Rate-Limits, das für den persönlichen E-Mail-Einsatz geeignet ist. Kostenpflichtige Stufen sind verfügbar, wenn Sie einen höheren Durchsatz benötigen.

Einrichtung

1. ThunderAI-Optionen öffnen

Klicken Sie in Thunderbird auf das ThunderAI-Menü und wählen Sie Options, oder gehen Sie zu
Extras → Add-ons und Themes → ThunderAI → Einstellungen.

2. Google Gemini-Integration auswählen

Wählen Sie im Bereich Connection die Option Google Gemini aus dem Integrations-Dropdown.

3. API-Schlüssel eingeben

Fügen Sie Ihren API-Schlüssel in das Feld API Key ein.

4. Modell auswählen

Wählen Sie das Gemini-Modell aus dem Dropdown Model. Klicken Sie auf die Schaltfläche, um die Liste zu aktualisieren.

5. (Optional) System Instructions konfigurieren

System Instructions ermöglichen es Ihnen, einen persistenten Kontext für alle Ihre Prompts festzulegen. Zum Beispiel, dem Modell anzuweisen, immer formell zu antworten, in einer bestimmten Sprache oder in einer bestimmten Rolle zu agieren.
Geben Sie Ihre Anweisungen in das Feld System Instructions auf der Optionsseite ein.

6. (Optional) thinkingBudget konfigurieren

Die Option thinkingBudget steuert, wie viel internes Reasoning das Modell vor der Antwort durchführt. Ein höherer Wert kann die Qualität bei komplexen Aufgaben verbessern, kostet aber mehr Zeit und Kontingent. Belassen Sie den Standardwert, sofern Sie keine besonderen Anforderungen haben.

7. Speichern und testen

Klicken Sie auf Save. Öffnen Sie eine beliebige E-Mail, verwenden Sie das ThunderAI-Menü und führen Sie einen Prompt aus, um die Verbindung zu überprüfen.

Tipps

  • Das kostenlose Kontingent wird monatlich erneuert und ist in der Regel für den persönlichen Gebrauch ausreichend.
  • System Instructions sind besonders nützlich, wenn Sie allgemeine Regeln für Ihren E-Mail-Assistenten festlegen möchten.

Fehlerbehebung

Fehler „API key not valid"

Prüfen Sie, ob der Schlüssel korrekt kopiert wurde. Generieren Sie bei Bedarf einen neuen Schlüssel aus AI Studio.

„Model not found" oder leere Modellliste

Stellen Sie sicher, dass Ihr API-Schlüssel Zugang zu Gemini-Modellen hat. Einige in eingeschränkten Google Cloud-Projekten erstellte Schlüssel haben möglicherweise begrenzten Modellzugang.
