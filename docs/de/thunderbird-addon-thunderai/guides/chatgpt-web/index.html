"ThunderAI" Einrichtungsanleitung - ChatGPT Web Interface

Die Integration mit der ChatGPT Web Interface ermöglicht es Ihnen, ThunderAI ohne API-Schlüssel zu nutzen.
ThunderAI interagiert direkt mit der ChatGPT-Website in Ihrem Namen, sodass ein kostenloses ChatGPT-Konto ausreicht.

API-Schlüssel erforderlichNein
Kostenlose OptionJa [Auch mit einem kostenlosen ChatGPT-Konto]
Geeignet fürSchneller Einstieg, keine Abrechnungseinrichtung

Voraussetzungen

  • Ein kostenloses (oder kostenpflichtiges) Konto auf chatgpt.com
  • Thunderbird mit installiertem ThunderAI

Einrichtung

1. ThunderAI-Optionen öffnen

Klicken Sie in Thunderbird auf das ThunderAI-Menü und wählen Sie Options, oder gehen Sie zu
Extras → Add-ons und Themes → ThunderAI → Einstellungen.

2. ChatGPT Web Interface auswählen

Wählen Sie im Bereich Connection die Option ChatGPT Web Interface aus dem Integrations-Dropdown.

3. Erforderliche Berechtigung erteilen

ThunderAI benötigt die Berechtigung, auf chatgpt.com zuzugreifen.

Diese Berechtigung ist optional und wird ausschließlich dazu verwendet, in Ihrem Namen mit der ChatGPT-Website zu interagieren. ThunderAI erfasst oder speichert keine Ihrer Daten.

4. Bei ChatGPT in Thunderbird anmelden

Dies müssen Sie nur einmal tun; Thunderbird hält die Sitzung aktiv.
Verwenden Sie die Schaltfläche auf der ThunderAI-Einstellungsseite, um einen neuen Thunderbird-Tab zu öffnen und chatgpt.com zu besuchen. Melden Sie sich mit Ihrem Konto an.

5. Integration testen

Öffnen Sie eine beliebige E-Mail, klicken Sie auf die ThunderAI-Schaltfläche in der Symbolleiste und wählen Sie einen Prompt (z. B. Summarize). ThunderAI öffnet ein Panel und interagiert mit ChatGPT.

Tipps und bekannte Einschränkungen

  • Die Antwortgeschwindigkeit hängt von der ChatGPT-Website ab und kann variieren. Die API-Integration ist in der Regel schneller und zuverlässiger.
  • Kostenlose Konten unterliegen den eigenen Nutzungslimits von ChatGPT (Rate-Limits, Modellverfügbarkeit).
  • Wenn ChatGPT das Layout seiner Website ändert, kann die Integration vorübergehend aufhören zu funktionieren, bis ThunderAI aktualisiert wird. Prüfen Sie die Statusseite auf bekannte Probleme.
  • Sie können die Verwendung eines Custom GPT oder Projekts global in den Einstellungen oder in der Konfiguration eines einzelnen Prompts festlegen.

Fehlerbehebung

ThunderAI zeigt einen Fehler oder keine Antwort

Stellen Sie sicher, dass Sie in einem Thunderbird-Tab bei ChatGPT angemeldet sind. Versuchen Sie, chatgpt.com direkt aus Thunderbird heraus aufzurufen, und prüfen Sie, ob die Sitzung aktiv ist. Verwenden Sie dafür die Schaltfläche auf der ThunderAI-Einstellungsseite.

Antworten sind langsam oder laufen ab

Dies ist in der Regel ein vorübergehendes Problem mit der ChatGPT-Website. Warten Sie einen Moment und versuchen Sie es erneut, oder wechseln Sie zur Integration OpenAI API für zuverlässigere Leistung.
