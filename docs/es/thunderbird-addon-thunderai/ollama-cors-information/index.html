"ThunderAI" Thunderbird Addon - Información CORS de Ollama


La configuración CORS de Ollama depende de su sistema operativo y de cómo ejecuta el servidor local.
En esta página encontrará toda la información recopilada con la ayuda de los usuarios de ThunderAI.
Si desea contribuir con una nueva configuración, por favor envíe un nuevo ticket.

Si no puede configurar CORS en el servidor o no tiene acceso a él, ahora puede conceder el permiso "Todas las URLs" a ThunderAI para evitar cualquier problema CORS.
Use el botón correspondiente en la página de opciones, como se muestra a continuación.

Docker Desktop

Al ejecutar un nuevo contenedor, abra "Configuración opcional" y añada una "Variable de entorno" con los siguientes valores:
  • Variable: OLLAMA_ORIGINS
  • Valor: moz-extension://*

Docker en línea de comandos

Al ejecutar un nuevo contenedor, añada esta opción al comando docker:
-e 'OLLAMA_ORIGINS=moz-extension://*'

Docker Compose

Gracias a Amras2322

Al configurar el servicio Ollama, añada la opción OLLAMA_ORIGINS en la sección environment del archivo docker-compose.yml:
services:
  ollama:
    image: ollama/ollama:latest
    container_name: ollama
        [...]
    environment:
    - OLLAMA_ORIGINS=moz-extension://*

Ollama en Windows

Gracias a actXc

  1. Abrir Configuración de Windows (tecla Windows + I)
  2. Buscar "variables de entorno"
  3. Hacer clic en "Editar las variables de entorno del sistema"
  4. En la nueva ventana, hacer clic en "Variables de entorno"
  5. Bajo "Variables del sistema", hacer clic en "Nuevo"
  6. Nombre de la variable: OLLAMA_ORIGINS
  7. Valor de la variable: moz-extension://*
  8. Confirmar con "Aceptar"
  9. Reiniciar el PC

Ollama en Linux

Gracias a lakonis

### Edición de /etc/systemd/system/ollama.service.d/override.conf
### Todo lo que esté entre aquí y el comentario de abajo se convertirá en el contenido del archivo drop-in

[Service]
Environment="OLLAMA_ORIGINS=moz-extension://*"

### Las ediciones por debajo de este comentario serán descartadas
[...]
Probado en Linux Manjaro

Ollama en macOS

Gracias a jatoko

  1. Abrir la terminal
  2. Ejecutar el siguiente comando en la terminal:
    launchctl setenv OLLAMA_ORIGINS "moz-extension://*"
  3. (Re)iniciar Ollama (p.ej. salir y volver a abrir)
