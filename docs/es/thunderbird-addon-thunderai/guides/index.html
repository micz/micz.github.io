Consulta nuestro tutorial para aprender a crear tu primer prompt personalizado.

Integraciones de IA

Conecta ThunderAI al backend de IA de tu elección. ¿No sabes cuál elegir? Consulta la tabla comparativa a continuación.

  • ChatGPT Web Interface — No se requiere clave de API; funciona también con una cuenta gratuita de ChatGPT.
  • OpenAI API — Conexión directa a los modelos de ChatGPT mediante clave de API.
  • Google Gemini — Conecta con los modelos Gemini de Google; nivel gratuito disponible.
  • Claude API — Los modelos Claude de Anthropic, ideales para el análisis detallado de correos.
  • Ollama (IA local) — Ejecuta la IA completamente en tu máquina. Privado, gratuito, sin necesidad de internet.
  • OpenAI Compatible API — LM Studio, DeepSeek, Mistral, OpenRouter y más.
