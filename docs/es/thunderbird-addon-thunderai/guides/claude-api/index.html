"ThunderAI" Guía de configuración - Claude API

La integración con la Claude API conecta ThunderAI con los modelos de IA Claude de Anthropic.
Claude es conocido por producir respuestas cuidadosas y bien estructuradas, y por manejar eficazmente documentos largos — ideal para leer y redactar correos electrónicos detallados.

Requisitos previos

  • Una cuenta de Anthropic con acceso a la API
  • Una clave de API de Claude válida
  • Thunderbird con ThunderAI instalado

Cómo obtener una clave de API

  1. Ve a console.anthropic.com/settings/keys
  2. Haz clic en Create Key
  3. Dale un nombre y copia la clave inmediatamente; no se mostrará de nuevo

Configuración

1. Abre las Opciones de ThunderAI

En Thunderbird, haz clic en el menú de ThunderAI y selecciona Options, o ve a
Herramientas → Complementos y temas → ThunderAI → Preferencias.

2. Selecciona la integración Claude API

En la sección Connection, elige Claude API del menú desplegable de integración.

3. Introduce tu clave de API

Pega tu clave de API de Claude en el campo API Key.

4. Concede el permiso requerido

La Claude API requiere que ThunderAI acceda al dominio anthropic.com.
Aparecerá un botón en la página de opciones para conceder este permiso — haz clic en él y confirma en el diálogo de Thunderbird.

Este es un permiso opcional que se solicita únicamente cuando seleccionas la integración de Claude. Permite que ThunderAI se comunique con los servidores de la API de Anthropic. ThunderAI no recopila ningún dato.

5. Elige un modelo

Selecciona un modelo de Claude del menú desplegable Model. Haz clic en el botón para actualizar la lista.

6. Guarda y prueba

Haz clic en Save. Abre cualquier correo electrónico, usa el menú de ThunderAI y ejecuta un prompt para confirmar que la integración funciona.

Consejos

  • Los modelos Claude manejan bien grandes cantidades de texto gracias a su amplia ventana de contexto, lo que resulta útil al procesar hilos de correo largos.
  • Claude tiende a ser exhaustivo en sus respuestas. Si prefieres respuestas más cortas, añade instrucciones como "Sé conciso" directamente en tus prompts o en una plantilla de prompt personalizado.
  • Supervisa el uso de tu API y los costes en console.anthropic.com.

Solución de problemas

"Authentication error" o "Invalid API key"

Verifica que la clave fue copiada correctamente. Si es necesario, genera una nueva desde la consola de Anthropic.

Error CORS o solicitudes fallidas

Asegúrate de haber concedido el permiso al dominio anthropic.com tal como se describe en el paso 4. Sin él, ThunderAI no puede llegar a la Claude API.

Error "Credit balance too low"

Añade créditos a tu cuenta de Anthropic en console.anthropic.com/settings/billing.

No aparecen modelos en el menú desplegable

Verifica que tu clave de API esté guardada y que se haya concedido el permiso al dominio. Intenta guardar la página de opciones y volver a abrirla.
