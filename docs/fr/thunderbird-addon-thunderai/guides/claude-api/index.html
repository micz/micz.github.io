Guide de configuration pour "ThunderAI" - Claude API

L'intégration Claude API connecte ThunderAI aux modèles AI Claude d'Anthropic.
Claude est reconnu pour produire des réponses soignées et bien structurées, et pour gérer efficacement les longs documents — idéal pour lire et rédiger des e-mails détaillés.

Clé API requiseOui
Option gratuiteNon [Facturation à l'usage]
Idéal pourAnalyses détaillées, e-mails longs, rédaction structurée

Prérequis

  • Un compte Anthropic avec accès à l'API
  • Une clé Claude API valide
  • Thunderbird avec ThunderAI installé

Obtenir une clé API

  1. Accédez à console.anthropic.com/settings/keys
  2. Cliquez sur Create Key
  3. Donnez-lui un nom et copiez la clé immédiatement, elle ne sera plus affichée

Configuration

1. Ouvrir les options de ThunderAI

Dans Thunderbird, cliquez sur le menu ThunderAI et sélectionnez Options, ou accédez à
Outils → Modules complémentaires et thèmes → ThunderAI → Préférences.

2. Sélectionner l'intégration Claude API

Dans la section Connection, choisissez Claude API dans le menu déroulant des intégrations.

3. Saisir votre clé API

Collez votre clé Claude API dans le champ API Key.

4. Accorder la permission requise

La Claude API requiert que ThunderAI accède au domaine anthropic.com.
Un bouton apparaîtra sur la page des options pour accorder cette permission — cliquez dessus et confirmez dans la boîte de dialogue Thunderbird.

Il s'agit d'une permission optionnelle demandée uniquement lorsque vous sélectionnez l'intégration Claude. Elle permet à ThunderAI de communiquer avec les serveurs API d'Anthropic. ThunderAI ne collecte aucune donnée.

5. Choisir un modèle

Sélectionnez un modèle Claude dans le menu déroulant Model. Cliquez sur le bouton pour actualiser la liste.

6. Enregistrer et tester

Cliquez sur Save. Ouvrez n'importe quel e-mail, utilisez le menu ThunderAI et exécutez un prompt pour confirmer que l'intégration fonctionne.

Conseils

  • Les modèles Claude gèrent de grandes quantités de texte grâce à leur large fenêtre contextuelle, utile pour traiter de longues conversations par e-mail.
  • Claude tend à être exhaustif dans ses réponses. Si vous préférez des résultats plus courts, ajoutez des instructions comme « Sois concis » directement dans vos prompts ou dans un modèle de prompt personnalisé.
  • Surveillez votre utilisation et vos coûts sur console.anthropic.com.

Dépannage

« Authentication error » ou « Invalid API key »

Vérifiez que la clé a été copiée correctement. Si nécessaire, régénérez une nouvelle clé depuis la console Anthropic.

Erreur CORS ou requêtes échouées

Assurez-vous d'avoir accordé la permission pour le domaine anthropic.com comme décrit à l'étape 4. Sans elle, ThunderAI ne peut pas atteindre la Claude API.

Erreur « Credit balance too low »

Ajoutez des crédits à votre compte Anthropic sur console.anthropic.com/settings/billing.

Aucun modèle n'apparaît dans le menu déroulant

Vérifiez que votre clé API est enregistrée et que la permission de domaine a été accordée. Essayez d'enregistrer la page des options et de la rouvrir.
