Guide de configuration pour "ThunderAI" - Google Gemini

L'intégration Google Gemini connecte ThunderAI aux modèles AI Gemini de Google via l'API Gemini.
Elle inclut la prise en charge d'options avancées telles que les System Instructions et le thinkingBudget, utiles pour les utilisateurs qui souhaitent un contrôle plus fin du comportement du modèle.

Clé API requiseOui
Option gratuiteOui [Niveau gratuit généreux disponible]
Idéal pourLes utilisateurs de l'écosystème Google ou souhaitant une option API gratuite

Prérequis

  • Un compte Google
  • Une clé Gemini API (niveau gratuit disponible)
  • Thunderbird avec ThunderAI installé

Obtenir une clé API

  1. Accédez à aistudio.google.com/apikey
  2. Cliquez sur Create API key
  3. Sélectionnez un projet Google Cloud existant ou créez-en un nouveau
  4. Copiez la clé API générée

L'API Gemini propose un niveau gratuit avec des limites généreuses, adapté à un usage personnel. Des niveaux payants sont disponibles si vous avez besoin d'un débit plus élevé.

Configuration

1. Ouvrir les options de ThunderAI

Dans Thunderbird, cliquez sur le menu ThunderAI et sélectionnez Options, ou accédez à
Outils → Modules complémentaires et thèmes → ThunderAI → Préférences.

2. Sélectionner l'intégration Google Gemini

Dans la section Connection, choisissez Google Gemini dans le menu déroulant des intégrations.

3. Saisir votre clé API

Collez votre clé API dans le champ API Key.

4. Choisir un modèle

Sélectionnez le modèle Gemini dans le menu déroulant Model. Cliquez sur le bouton pour actualiser la liste.

5. (Optionnel) Configurer les System Instructions

Les System Instructions vous permettent de définir un contexte persistant pour tous vos prompts. Par exemple, demander au modèle de toujours répondre formellement, dans une langue spécifique, ou d'adopter un certain rôle.
Saisissez vos instructions dans le champ System Instructions sur la page des options.

6. (Optionnel) Configurer le thinkingBudget

L'option thinkingBudget contrôle la quantité de raisonnement interne que le modèle effectue avant de répondre. Une valeur plus élevée peut améliorer la qualité sur des tâches complexes au détriment de la vitesse et du quota. Laissez la valeur par défaut sauf si vous avez des besoins spécifiques.

7. Enregistrer et tester

Cliquez sur Save. Ouvrez n'importe quel e-mail, utilisez le menu ThunderAI et exécutez un prompt pour vérifier la connexion.

Conseils

  • Le niveau gratuit se renouvelle mensuellement et est généralement suffisant pour un usage personnel.
  • Les System Instructions sont particulièrement utiles si vous devez spécifier des règles générales pour votre assistant e-mail.

Dépannage

Erreur « API key not valid »

Vérifiez que la clé a été copiée correctement. Si nécessaire, régénérez une nouvelle clé depuis AI Studio.

« Model not found » ou liste de modèles vide

Assurez-vous que votre clé API a accès aux modèles Gemini. Certaines clés créées dans des projets Google Cloud restreints peuvent avoir un accès limité aux modèles.
