Guide de configuration pour "ThunderAI" - ChatGPT Web Interface

L'intégration ChatGPT Web Interface vous permet d'utiliser ThunderAI sans clé API.
ThunderAI interagit directement avec le site ChatGPT en votre nom, un compte ChatGPT gratuit suffit donc.

Clé API requiseNon
Option gratuiteOui [Même avec un compte ChatGPT gratuit]
Idéal pourDémarrer rapidement, sans configuration de facturation

Prérequis

  • Un compte gratuit (ou payant) sur chatgpt.com
  • Thunderbird avec ThunderAI installé

Configuration

1. Ouvrir les options de ThunderAI

Dans Thunderbird, cliquez sur le menu ThunderAI et sélectionnez Options, ou accédez à
Outils → Modules complémentaires et thèmes → ThunderAI → Préférences.

2. Sélectionner la ChatGPT Web Interface

Dans la section Connection, choisissez ChatGPT Web Interface dans le menu déroulant des intégrations.

3. Accorder la permission requise

ThunderAI a besoin de la permission d'accéder à chatgpt.com.

Cette permission est optionnelle et utilisée uniquement pour interagir avec le site ChatGPT en votre nom. ThunderAI ne collecte ni ne stocke aucune de vos données.

4. Se connecter à ChatGPT dans Thunderbird

Vous ne devez le faire qu'une seule fois ; Thunderbird maintiendra la session active.
Utilisez le bouton dans la page des préférences de ThunderAI pour ouvrir un nouvel onglet Thunderbird et visiter chatgpt.com. Connectez-vous avec votre compte.

5. Tester l'intégration

Ouvrez n'importe quel e-mail, cliquez sur le bouton ThunderAI dans la barre d'outils et sélectionnez un prompt (ex. : Summarize). ThunderAI ouvrira un panneau et interagira avec ChatGPT.

Conseils et limitations connues

  • La vitesse de réponse dépend du site ChatGPT et peut varier. L'intégration via API est généralement plus rapide et plus fiable.
  • Les comptes gratuits sont soumis aux limites d'utilisation de ChatGPT (rate limits, disponibilité des modèles).
  • Si ChatGPT modifie la mise en page de son site, l'intégration peut cesser de fonctionner temporairement jusqu'à la mise à jour de ThunderAI. Consultez la page de statut pour tout problème connu.
  • Vous pouvez configurer l'utilisation d'un Custom GPT ou d'un Projet globalement dans les préférences ou dans la configuration d'un prompt spécifique.

Dépannage

ThunderAI affiche une erreur ou aucune réponse

Assurez-vous d'être connecté à ChatGPT dans un onglet Thunderbird. Essayez de visiter chatgpt.com directement depuis Thunderbird et vérifiez que la session est active. Utilisez le bouton dans la page des préférences de ThunderAI.

Les réponses sont lentes ou expirent

Il s'agit généralement d'un problème temporaire avec le site ChatGPT. Attendez un moment et réessayez, ou envisagez de passer à l'intégration OpenAI API pour des performances plus fiables.
