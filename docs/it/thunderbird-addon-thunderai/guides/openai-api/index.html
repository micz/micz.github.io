Guida di configurazione per "ThunderAI" - OpenAI API

L'integrazione con OpenAI API connette ThunderAI direttamente ai server di OpenAI tramite una chiave API.
Rispetto all'interfaccia web, è più veloce, più affidabile e ti offre il controllo esplicito sul modello da utilizzare.

Chiave API richiestaSì
Opzione gratuitaNo [Pagamento a consumo]
Ideale perAccesso stabile e veloce ai modelli ChatGPT

Prerequisiti

  • Un account OpenAI con accesso alle API abilitato
  • Una chiave API valida
  • Thunderbird con ThunderAI installato

Nota: l'utilizzo dell'API OpenAI viene fatturato separatamente rispetto all'abbonamento ChatGPT. Anche se hai ChatGPT Plus, devi comunque aggiungere crediti al tuo account API.

Ottenere una chiave API

  1. Vai su platform.openai.com/api-keys
  2. Clicca su Create new secret key
  3. Assegnagli un nome (ad es. "ThunderAI") e clicca su Create
  4. Copia subito la chiave: non verrà mostrata di nuovo

Configurazione

1. Apri le opzioni di ThunderAI

In Thunderbird, clicca sul menu ThunderAI e seleziona Options, oppure vai su
Strumenti → Componenti aggiuntivi e temi → ThunderAI → Preferenze.

2. Seleziona l'integrazione OpenAI API

Nella sezione Connection, scegli OpenAI API dal menu a tendina delle integrazioni.

3. Inserisci la tua chiave API

Incolla la tua chiave API nel campo API Key. La chiave viene memorizzata localmente in Thunderbird e non viene mai inviata ad altri server oltre a quelli di OpenAI.

4. Scegli un modello

Seleziona il modello che desideri utilizzare dal menu a tendina Model. Clicca sul pulsante per aggiornare l'elenco.

5. Salva e testa

Clicca su Save. Apri una qualsiasi email, usa il menu ThunderAI ed esegui un prompt per verificare che tutto funzioni.

Suggerimenti

  • L'utilizzo delle API viene misurato in token. Le email sono generalmente brevi, quindi i costi sono di solito molto bassi. Controlla il tuo utilizzo su platform.openai.com/usage.
  • OpenAI aggiunge regolarmente nuovi modelli. Aggiorna l'elenco dei modelli nelle opzioni di ThunderAI per vedere gli ultimi disponibili.

Risoluzione dei problemi

Errore "Invalid API key"

Verifica di aver copiato la chiave correttamente: non devono esserci spazi extra. Se la chiave continua a non funzionare, generane una nuova dalla piattaforma OpenAI.

Errore "You exceeded your current quota"

Il tuo account API OpenAI ha esaurito i crediti. Aggiungine altri su platform.openai.com/settings/organization/billing.

Nessun modello nel menu a tendina

Assicurati che la tua chiave API sia salvata e valida. Se il problema persiste, controlla la connessione internet.
