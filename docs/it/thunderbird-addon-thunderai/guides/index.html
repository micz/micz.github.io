Guide passo dopo passo per aiutarti a sfruttare al meglio ThunderAI.
Che tu stia iniziando ora o voglia configurare una specifica integrazione AI, qui trovi tutto il necessario.
Consulta il nostro tutorial per imparare a creare il tuo primo prompt personalizzato.

Integrazioni AI

Collega ThunderAI al backend AI di tua scelta. Non sai quale scegliere? Consulta la tabella comparativa qui sotto.

  • ChatGPT Web Interface — Nessuna chiave API richiesta, funziona anche con un account ChatGPT gratuito.
  • OpenAI API — Connessione diretta ai modelli ChatGPT tramite chiave API.
  • Google Gemini — Connettiti ai modelli Gemini di Google, livello gratuito disponibile.
  • Claude API — I modelli Claude di Anthropic, ottimi per l'analisi dettagliata delle email.
  • Ollama (AI locale) — Esegui l'AI interamente sul tuo computer. Privato, gratuito, senza internet.
  • OpenAI Compatible API — LM Studio, DeepSeek, Mistral, OpenRouter e altri.
