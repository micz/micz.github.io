Guida di configurazione per "ThunderAI" - ChatGPT Web Interface

L'integrazione con la ChatGPT Web Interface ti consente di usare ThunderAI senza una chiave API.
ThunderAI interagisce direttamente con il sito ChatGPT per tuo conto, quindi è sufficiente un account ChatGPT gratuito.

Chiave API richiestaNo
Opzione gratuitaSì [Anche con un account ChatGPT gratuito]
Ideale perIniziare subito, senza configurazioni di pagamento

Prerequisiti

  • Un account gratuito (o a pagamento) su chatgpt.com
  • Thunderbird con ThunderAI installato

Configurazione

1. Apri le opzioni di ThunderAI

In Thunderbird, clicca sul menu ThunderAI e seleziona Options, oppure vai su
Strumenti → Componenti aggiuntivi e temi → ThunderAI → Preferenze.

2. Seleziona la ChatGPT Web Interface

Nella sezione Connection, scegli ChatGPT Web Interface dal menu a tendina delle integrazioni.

3. Concedi il permesso richiesto

ThunderAI necessita del permesso per accedere a chatgpt.com.

Questo permesso è opzionale e viene utilizzato esclusivamente per interagire con il sito ChatGPT per tuo conto. ThunderAI non raccoglie né memorizza alcun dato.

4. Accedi a ChatGPT in Thunderbird

Devi farlo solo una volta; Thunderbird manterrà la sessione attiva.
Usa il pulsante nella pagina delle preferenze di ThunderAI per aprire una nuova scheda Thunderbird e visitare chatgpt.com. Accedi con il tuo account.

5. Testa l'integrazione

Apri una qualsiasi email, clicca sul pulsante ThunderAI nella barra degli strumenti e seleziona un prompt (ad es. Summarize). ThunderAI aprirà un pannello e interagirà con ChatGPT.

Suggerimenti e limitazioni note

  • La velocità di risposta dipende dal sito ChatGPT e può variare. L'integrazione via API è generalmente più veloce e affidabile.
  • Gli account gratuiti sono soggetti ai limiti di utilizzo di ChatGPT (rate limit, disponibilità dei modelli).
  • Se ChatGPT modifica il layout del suo sito, l'integrazione potrebbe smettere di funzionare temporaneamente fino all'aggiornamento di ThunderAI. Consulta la pagina di stato per eventuali problemi noti.
  • Puoi configurare l'uso di un Custom GPT o di un Progetto sia globalmente nelle preferenze sia nella configurazione del singolo prompt.

Risoluzione dei problemi

ThunderAI mostra un errore o nessuna risposta

Assicurati di essere connesso a ChatGPT in una scheda Thunderbird. Prova a visitare direttamente chatgpt.com da Thunderbird e verifica che la sessione sia attiva. Usa il pulsante nella pagina delle preferenze di ThunderAI.

Le risposte sono lente o vanno in timeout

Di solito si tratta di un problema temporaneo con il sito ChatGPT. Attendi un momento e riprova, oppure considera di passare all'integrazione OpenAI API per prestazioni più affidabili.
