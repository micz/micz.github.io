Guida di configurazione per "ThunderAI" - Google Gemini

L'integrazione con Google Gemini connette ThunderAI ai modelli AI Gemini di Google tramite la Gemini API.
Include il supporto per opzioni avanzate come System Instructions e thinkingBudget, utili per gli utenti che desiderano maggiore controllo sul comportamento del modello.

Chiave API richiestaSì
Opzione gratuitaSì [Livello gratuito generoso disponibile]
Ideale perChi preferisce l'ecosistema Google o vuole un'opzione API gratuita

Prerequisiti

  • Un account Google
  • Una chiave Gemini API (livello gratuito disponibile)
  • Thunderbird con ThunderAI installato

Ottenere una chiave API

  1. Vai su aistudio.google.com/apikey
  2. Clicca su Create API key
  3. Seleziona un progetto Google Cloud esistente o creane uno nuovo
  4. Copia la chiave API generata

La Gemini API offre un livello gratuito con limiti generosi, adatto all'uso personale delle email. Sono disponibili piani a pagamento se hai bisogno di maggiore throughput.

Configurazione

1. Apri le opzioni di ThunderAI

In Thunderbird, clicca sul menu ThunderAI e seleziona Options, oppure vai su
Strumenti → Componenti aggiuntivi e temi → ThunderAI → Preferenze.

2. Seleziona l'integrazione Google Gemini

Nella sezione Connection, scegli Google Gemini dal menu a tendina delle integrazioni.

3. Inserisci la tua chiave API

Incolla la tua chiave API nel campo API Key.

4. Scegli un modello

Seleziona il modello Gemini dal menu a tendina Model. Clicca sul pulsante per aggiornare l'elenco.

5. (Opzionale) Configura le System Instructions

Le System Instructions ti permettono di impostare un contesto persistente per tutti i tuoi prompt. Ad esempio, puoi indicare al modello di rispondere sempre in modo formale, in una lingua specifica, o di assumere un determinato ruolo.
Inserisci le istruzioni nel campo System Instructions nella pagina delle opzioni.

6. (Opzionale) Configura il thinkingBudget

L'opzione thinkingBudget controlla quanta elaborazione interna il modello esegue prima di rispondere. Un valore più alto può migliorare la qualità su compiti complessi, a scapito della velocità e della quota. Lascia il valore predefinito, a meno che tu non abbia esigenze specifiche.

7. Salva e testa

Clicca su Save. Apri una qualsiasi email, usa il menu ThunderAI ed esegui un prompt per verificare la connessione.

Suggerimenti

  • Il livello gratuito si rinnova mensilmente ed è generalmente sufficiente per l'uso personale.
  • Le System Instructions sono particolarmente utili se hai bisogno di specificare regole generali per il tuo assistente email.

Risoluzione dei problemi

Errore "API key not valid"

Verifica che la chiave sia stata copiata correttamente. Se necessario, rigenera una nuova chiave da AI Studio.

"Model not found" o elenco modelli vuoto

Assicurati che la tua chiave API abbia accesso ai modelli Gemini. Alcune chiavi create in progetti Google Cloud con restrizioni potrebbero avere accesso limitato ai modelli.
