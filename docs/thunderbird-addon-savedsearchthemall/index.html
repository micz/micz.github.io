Thunderbird Addon: SavedSearchThemAll!

Given this Thunderbird bug (yes bug, not enhancement!), I developed this addon to avoid the problem (I was going to lose too much time updating manually every search folder).


How does it work?
It adds a menu item named "Update search folders target" on the tools menu, just click on it and all the saved search folders will search on all folders you want.

There are also two options, set them before using the addon:

  • Local Folders' saved search folders will search on all folders of all accounts.
  • Add only subfolders of currently selected folders of all accounts.
If you check no options all the saved search folders will search on all folders (of the same account).
