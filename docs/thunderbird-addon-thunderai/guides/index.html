AI Integrations

Connect ThunderAI to the AI backend of your choice. Not sure which one to pick? See the comparison table below.

  • ChatGPT Web Interface — No API key needed, works also with a free ChatGPT account.
  • OpenAI API — Direct connection to ChatGPT models via API key.
  • Google Gemini — Connect to Google's Gemini models, free tier available.
  • Claude API — Anthropic's Claude models, great for detailed email analysis.
  • Ollama (Local AI) — Run AI entirely on your machine. Private, free, no internet required.
  • OpenAI Compatible API — LM Studio, DeepSeek, Mistral, OpenRouter, and more.
